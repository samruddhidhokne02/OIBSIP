# Voice Assistant

## Description
This project is a Python-based voice assistant that listens to user voice commands and performs different tasks.
It can respond with greetings, tell the current time and date, and search topics on the web.

## Features
- Captures voice input using microphone
- Responds to "Hello" with a greeting
- Tells current time and date
- Performs web search on user command
- Uses text-to-speech for speaking responses
- Handles errors if the voice is not recognized

## Technologies Used
- Python
- speech_recognition
- pyttsx3
- datetime
- webbrowser

## How to Run
1. Install required libraries:
pip install SpeechRecognition pyttsx3 pyaudio

2. Run the Python file:
python voice_assistant.py

3. Speak commands like:
- Hello
- What is the time?
- What is today's date?
- Search Python programming

## Output
The assistant listens to the user's voice and gives spoken responses.
